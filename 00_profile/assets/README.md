# Profile assets

Put your source files here, then the site references copies in `01_www/assets/`:

- `profile.jpg` — square headshot (≥ 500×500). Copied to `01_www/assets/profile.jpg`.
- `CV.pdf` — your CV. Copied to `01_www/assets/CV.pdf` (linked by the download button).

The `site-engineer` copies these into `01_www/assets/` during build. Keep originals here
so the deployable folder stays clean.
